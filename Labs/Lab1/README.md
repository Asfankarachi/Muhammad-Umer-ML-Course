## Overview

This lab introduced the foundational concepts of machine learning, including the distinction between supervised, unsupervised, and reinforcement learning. The goal was to build a working understanding of how machine learning models are trained, evaluated, and applied to real-world problems.

## What I Learned

Through this lab, I developed an understanding of how data is used to train predictive models. I learned about key concepts such as features, labels, training sets, and test sets.

## Challenges Faced

The primary challenge was understanding the difference between overfitting and underfitting, and how to interpret evaluation metrics such as accuracy, precision, and recall.

## Tools Used

Python, Jupyter Notebook, scikit-learn, pandas, matplotlib

## Key Takeaway

Machine learning is not just about building models — it is about understanding the data, choosing the right approach, and evaluating results meaningfully.
