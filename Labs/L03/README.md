## Overview

This lab explored how neurons encode, transmit, and process information, focusing on rate coding versus temporal coding and how populations of neurons collectively represent information.

## What I Built

I implemented a simulation of a feedforward neural layer processing input signals, modeling how synaptic weights determine which downstream neurons fire and how spike timing carries information.

## Tools Used

- Python 3
- NumPy
- Matplotlib for spike raster plots

## What I Learned

Rate coding assumes average firing rate carries the signal while temporal coding relies on precise spike timing. Temporal coding is more information-efficient but more sensitive to noise, which is why the brain uses both strategies depending on context.
